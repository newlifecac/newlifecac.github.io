旧约有39卷，新约有27卷。整本《圣经》有66卷。

简单说来，《圣经》就是约。
旧约是上帝跟以色列人立的约，
新约是上帝跟所有信耶稣的人立的约。   

旧约和新约的分界线是耶稣基督的降生。
耶稣基督降生之前是旧约时代，
耶稣基督降生之后是新约时代。
现在全世界通用的公元记年，
也是以耶稣基督的降生为分界线。
耶稣基督降生前是公元前，耶稣基督降生后是公元后。
今年是2023年，就是说，耶稣基督是2023年前降生的。

### [第二课: 上帝]
``` 
自古以来，人与其它动物的区别是，人知道有上帝，人想认识上帝。
但人靠自己不能认识真神，于是人就造出许多神。
中国有伏羲，盘古，女娲，玉皇大帝等各路神仙。
西方有宙斯为首的众神。
 
人凭着自己的认知能力不能认识上帝，
人不能凭自己的能力，判断力，认知力认识上帝。
因为人是罪人，罪的含义就是没有打中目标，
罪的结果就是与神隔绝。
罪人不认识上帝，才会拜一些人物，动物，植物，甚至石头，
泥土木头做的偶像，将它们当做神。
人想自己认识神，很像“瞎子摸象”。
 
从前有个印度寓言，瞎子摸象：
几个盲人坐在路边，听说有驯象师带着一头大象路过，
就想“看看”大象。一个盲人摸到大象肚子，
说，原来大象是一堵墙。另一个抱住大象腿，
说，大象是一棵大树。还有一个摸到大象耳朵，
说，大象是一把蒲扇。摸到大象鼻子的
说，大象是一条蛇……。
 
这个寓言告诉我们，罪人认知能力的局限性。
罪人因为自己的有限性，认识事物往往是片面的，狭隘的，局限的。
人认识事物尚且如此，更何况认识上帝。
 
神怜悯我们有这样的弱点和局限，就借着圣经启示我们，
告诉我们怎样认识神。
 
新约圣经的前四卷书是福音书：《
马太福音》，《马可福音》，《路加福音》，《约翰福音》。
这四卷书都是讲耶稣的生平和教导。
但四福音书分别给了四个不同角度，
让我们能够比较完整地认识耶稣基督，
这位成为肉身的上帝。
 
马太福音所告诉我们的耶稣基督是一位君王。形象上用狮子形象表达。
马可福音所告诉我们的耶稣基督是一位仆人。形象上用耕牛形象表达。
路加福音所告诉我们的耶稣基督是人。形象上用人形像表达。
约翰福音所告诉我们的耶稣基督是神。形象上用鹰形象表达。
 
在圣经里，我们所认识的上帝是
 
1. 圣父：创造天地万物的神
2. 圣子：拯救世人脱离死亡的神
3. 圣灵：让我们认罪，悔改归向神，
        并让我们明白一切真理的神
 
他们是一位，不是三个神。这方面我们人很难明白。
我们只需要知道，神是一位，圣父，圣子，圣灵是一体，
都是神，同尊，同荣，同受敬拜。
 
最后，帮助我们认识上帝，并可以清楚说明我们所信的上帝的，
是《使徒信经》。我们一定要熟记，牢记使徒信经。
 
后附《使徒信经全文》
-----------

5.  熟读，牢记《使徒信经》，并学会使用《使徒信经》来告知我们所信的上帝是怎样的上帝。

### [第三课: 救恩]
```
整本《圣经》的主题就是“救赎（救恩）”。
如果将整本《圣经》的内容浓缩为一节经文，
那就是我们每个人必须背诵的《约翰福音》第三章第16节（约3:16）。
约翰福音三章16节这样说：
---------
“神爱世人，甚至将他的独生子赐给他们，
叫一切信他的，不致灭亡，反得永生”。
---------
这节经文分解起来是：
1. 上帝跟人的关系是什么？“爱”。
2.  上帝爱世人，爱到什么程度？“舍弃自己的独生子”。
3. 上帝为什么这样爱？“让所有信他的不至灭亡，反得永生 ”。

以神的大能，救人很容易，不费吹灰之力，不必举手之劳。

以神的公义，人犯罪的结果（代价）就是死。

神不会违背他自己的公义的标准，不能用大能来破坏公义。
那么所有犯罪的人就必须死。

神除了公义和大能，神还有大爱。
这个爱就是舍己的爱。
神让自己的儿子代替所有信他的人被钉在十字架上。
满足上帝的公义（罪人必须死）和慈爱（上帝的儿子死，让信的人活）。
这就是救赎，付出昂贵的救恩。

新约《以弗所书》2:13-16节说的：
“你们从前远离神的人，如今却在基督耶稣里，靠着他的血，已经得亲近了。
因他使我们和睦（因他是我们的和睦），将两下合而为一，拆毁了中间隔断的墙；
而且以自己的身体废掉冤仇，就是那记在律法上的规条，
为要将两下藉着自己造成一个新人，如此便成就了和睦。
既在十字架上灭了冤仇，便藉这十字架使两下归为一体，与神和好了”。

我们所信的上帝是独一的，是独特的。因为：

1. 神是大能的，创造天地万物的主。
2. 神是公义的，圣洁的，绝不姑息罪的神。
3. 神是有大爱，为了爱信他的人，
   甚至愿意付出自己（独生子）生命为代价，救赎我们的神。

因神的救赎，我们脱离永远的死亡，脱离罪恶的权势。
我们有了新生命。这就是神的救赎。
------------
问题与思考：
1. 《圣经》的主题是什么？
2. 《圣经》的内容如果浓缩为一节经文，是哪一节？请背诵这节基督徒必知，考试必考的经文。
3.  按照神的公义，罪人的结局是什么？
4.  按照神的大爱，上帝为信他的罪人做了什么？
```

### [第五课: 圣礼]
```
在天主教会里，圣礼有七项，“婚礼”，“丧礼”等都列入圣礼。
 
基督教会的圣礼有两项，都是耶稣基督亲自设立的。
那就是“洗礼“和“圣餐礼“。
 
耶稣基督30岁出来工作的时候，第一件事就是接受洗礼。
他接受了施洗约翰的洗礼。施洗约翰的洗礼是“悔改的洗礼”，
耶稣基督是无罪的，不需要悔改。但耶稣基督还是受洗了，
这是耶稣基督要为基督徒设立榜样。
基督徒就是效法基督的，耶稣受洗了，基督徒也应该受洗。
耶稣基督立了榜样。
 
耶稣复活升天之前，向门徒传达了大使命：
往普天下去，传福音给万民，给他们施洗……。
耶稣基督下达了命令。并且耶稣基督还有应许说：信而受洗，必然得救。
 
一、基督徒为什么要洗礼？
1. 耶稣基督的榜样
2. 耶稣基督的命令
3. 耶稣基督的应许
 
后来门徒在传福音，建立教会的过程中都认真执行这个命令，
曾经有一次给三千人施洗。
 
二、洗礼仪式有什么（象征）意义呢？
 
洗礼这个中文翻译会给人错觉，错以为洗礼和“洗”有关系。
就望文生义说，洗礼就是洗净人的罪。
“洗礼“最佳中文翻译是“浸礼”，
就是在洗礼仪式中将人完全浸入水中。
人在水中，就是死亡。从水中被扶起来，就是复活。
因此，洗礼的象征意义实际上是“出死入生”。
基督徒的生命是出死入生的生命。
 
简单总结一下：

一、 为什么要洗礼？
1. 耶稣基督的榜样，基督徒要效法基督（榜样）
2. 耶稣基督的命令，教会必须给人洗礼（命令）
3. 耶稣基督的应许，信而受洗必然得救（应许）
 
二、洗礼有什么意义？ 
简答，出死入生。复活的新生命。
“我们藉着洗礼归入死，和他一同埋葬，
原是叫我们一举一动有新生的样式，
像基督藉着父的荣耀从死里复活一样”（新约《罗马书》6:4）。
-----------------
圣餐礼

洗礼和圣餐礼都不能顾名思义。
就是说，不要以为洗礼和洗有关系。
圣餐礼也不能理解为宴会吃喝。
 
圣餐的英文是U-Christ，或者 Holly Communion. 
都是“与基督神圣联合”的意思。
就是说，圣餐礼是纪念耶稣牺牲自己的生命，
献出他的血（圣餐礼中的葡萄汁）和肉（圣餐礼中的饼）
让我们借此与基督的生命相连接，
弟兄姐妹也借着圣餐彼此连接，成为一体。所以，圣餐的意义是：

 
1. 纪念耶稣基督为我们而死
2. 我们借着圣餐与基督的生命联合
 
因为耶稣基督是为罪人而死，担当罪人所有的罪。
因此，领受圣餐的人必须：省察并承认自己的罪，认罪悔改。
以谦卑，敬畏的心领受。如果不懂得圣餐意义，
以轻慢的态度吃喝饼和葡萄汁，
就是对神的冒犯。万万不可。
------------

### [第七课: 奉献]
```
在旧约圣经里，上帝向他的选民以色列人定规了“十一奉献”。

“十一奉献”就是各人要将每周收入的十分之一送到神的面前。
圣经称之为“当纳的”，就是必须的，没有任何推卸的。
只要有收入，就必须奉献十分之一。

“地上所有的，无论是地上的种子，是树上的果子，
十分之一是耶和华的，是归耶和华为圣的”。
旧约《利未记》27:30.

奉献是上帝给他的选民的命令，
也是圣经里唯一的，允许“试试看”的教导。

“万军之耶和华说：你们要将当纳的十分之一全然送入仓库，
使我家有粮，以此试试我，是否为你们敞开天上的窗户，
倾福与你们，甚至无处可容”。旧约《玛拉基书》3:10.  

为什么上帝的选民（以色列人和基督徒）要有“十一奉献”？

1. 上帝的命令 《利未记》 27:30
2.  耶稣的称赞  《马可福音》 12:41-44
3. 承认上帝的主权 《申命记》 14:28
4. 表达对上帝的感恩  《诗篇》116:12
5. 提醒我们，一切都是神的恩典 《申命记》8:18

基督徒对奉献应有的认识
1. 奉献不是捐款。捐款和施舍相同，捐款是帮助弱者。
   可以做，也可以不做。
2. 十一奉献是“当纳”的，是必须的。
3. 十一奉献可以使神的家，教会有经济预算开展事工。
   每个基督徒，教会的会员对教会，就是上帝的家，都负有责任。
4. 相信上帝的应许和恩赐
5. 圣经说，财宝在哪里，心就在哪里。
   如果你真的爱上帝，必不会空手来朝见神。

### [第八课: 四个属灵原则]
```
《圣经》告诉我们，上帝创造宇宙万物和人。
上帝创造宇宙，也设置了宇宙的运行规则。
比如我们能感知的四季轮转，日夜更替，潮起潮落。
上帝造人，属灵的活人。上帝也设置了一些属灵的定律定规着我们和神的关系:
（请打开你手上的《圣经》，让我们对照着查看）       

第一律：神爱世人。爱上帝的人，神在他的生命有奇妙的计划。     

1．神的爱：“神爱世人，甚至将他的独生子（耶稣）赐给他们，
叫一切信他的，不至灭亡，反得永生。” （《约翰福音》3 :16）     
2．神的计划：“耶稣说：“我来了是要叫人得生命，
并且得的更丰盛。”(有意义的生命） （《约翰福音》10:10）    
 
为什么大多数的人没有经验过这种丰盛的生命呢？那是因为……       

第二律：人因有罪而与神隔绝，
所以不能知道并经验神的爱和神为他生命的计划。   

 1. 人的罪：是荣耀的亏损。
 “因为世人都犯了罪，亏缺了神的荣耀.。” （《罗马书》3 : 23）     
2. 人的罪：是与神隔绝。
“因为罪的工价就是死。”（与永生上帝隔绝关系） （《罗马书》6: 23 ）   
     
罪人与神关系是隔绝的。人不断地用自己的方法试图恢复关系，
如善行、道德、宗教、哲学等，来寻求神和丰盛的生命，
但是总得不着。第三律告诉我们恢复关系的唯一方法       

第三律：耶稣基督是唯一的道路。
信他的人才能经验神的爱并知道神为你生命的计划。     

1．耶稣为我们死。“唯有基督在我们还作罪人的时侯为我们死，
神的爱就在此向我们显明了。”（《罗马书》5:8）   
 2．耶稣从死里复活。“基督照圣经所说，为我们的罪死了，
 而且埋葬了，又照《圣经》所说，第三天复活了，并且显给矶法看，
 然后显给十二使徒看，后来一时显给五百多弟兄看。”(《哥林多前书》15:3～6)     
3．耶稣基督是唯一的道路。耶稣说：“我就是道路、真理、生命；
若不籍着我，没有人能到父（神）那里去。”(《约翰福音》14:6）     

神差他的儿子耶稣基督为我们的罪死在十字架上，
成为神与人中间的桥梁。只知道以上三个定律还不够……       

第四律：我们必须自愿接受耶稣基督作救主和生命的主，
才能知道并经验神的爱和神为我们生命的计划     

1．我们必须愿意接受基督：“凡接待他的，就是信他名的人，
他就赐他们权柄做神的儿女。” (《约翰福音》1:12）     
2．我们藉着信心接受基督：“你们得救是本乎恩，也因着信，
这并不是出于自己，乃是神所赐的，也不是出于行为，免得有人自夸。”（《以弗所书》2:8，9）     
3．我们必须亲自接受耶稣基督进入心中。
耶稣说:：“看哪，我站在门外叩门，若有听见我声音就开门的，
我要进到他那里去。”（《启示录》3 : 20）

口里承认，心里愿意（相信），是最美好的关系。
愿意我们与神的关系就是这样的关系。
从心里愿意承认并接受耶稣基督成为我们生命的主，他本来就是。阿们！
